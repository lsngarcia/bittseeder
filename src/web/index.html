Let's Encrypt (Auto-Certificate)

When Domain + Email are set, BittSeeder obtains and auto-renews a Let's Encrypt certificate via HTTP-01 challenge. The cert/key path fields above are set automatically — no manual configuration required. The HTTP challenge port must be reachable from the internet on port 80 (use a reverse proxy or iptables redirect if needed).
